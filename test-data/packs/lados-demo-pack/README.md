# Lados Demo Pack

This is a safe manifest-only pack for Phase 18P registry testing.

It is used to verify:
- Publish Pack upload accepts `.ladosPack`.
- Registry review can verify the listing.
- Browse Registry can install a verified listing.
- Installed pack and node declarations appear in Lados.

The declared demo node has no runtime executor. Phase 18P intentionally does not execute uploaded bundle code.
